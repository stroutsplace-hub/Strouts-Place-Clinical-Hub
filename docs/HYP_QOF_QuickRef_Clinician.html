What Must Be Coded — BP Review

Use the (RP) LTC Review Template CEG v1.4 in EMIS. All items must be coded on the day of the appointment.

🩺  Clinical Measurements
  • Blood pressure (clinic)SNOMED BP Enter systolic and diastolic values separately
  • OR Home blood pressure monitoring averageHBPM code Must use HBPM-specific SNOMED code — generic BP code will not satisfy QOF
  • Weight & BMI
  • QRISK3 calculated & coded
📋  Lifestyle & Review Items
  • Smoking status
  • Alcohol status AUDIT-C if not recorded in past 12 months
  • Lifestyle advice — diet, salt reduction, exercise
  • Medication review — adherence, side-effects Pharmacist only
⚠️
HBPM coding is critical. An HBPM reading coded with a standard clinic BP SNOMED code will not be recognised as satisfying the QOF indicator. Always use the HBPM-specific code.
🚨
BP ≥180/120 mmHg — inform the GP same day. Do not discharge the patient until the GP has been made aware. The GP will assess for emergency features and determine whether same-day referral is required.
🔄
After any medication change — arrange a repeat BP check within 4–8 weeks and re-code the result. The most recent reading in the window counts.

When to Refer — NICE NG136

🚨  Same-Day Emergency Referral

BP ≥180/120 mmHg with any of the following:

  • →Signs of retinal haemorrhage or papilloedema (accelerated hypertension)
  • →Life-threatening symptoms — new confusion, chest pain, signs of heart failure or acute kidney injury
  • →Suspected phaeochromocytoma — labile/postural hypotension, headache, palpitations, pallor, diaphoresis
📋  Routine / Urgent Specialist Referral
  • →BP uncontrolled on optimal tolerated doses of 4 drugs (resistant hypertension)
  • →Age <40 with hypertension — specialist evaluation for secondary causes
  • →Signs or symptoms suggesting a secondary cause — renal bruits, hypokalaemia, Cushingoid features
  • →Postural hypotension persisting despite addressing likely causes
⚠️
BP ≥180/120 mmHg without emergency features — do not refer immediately. Investigate for target organ damage (fundoscopy, urinalysis, renal function, ECG) as soon as possible. If target organ damage found: start treatment immediately. If not: repeat clinic BP within 7 days.
