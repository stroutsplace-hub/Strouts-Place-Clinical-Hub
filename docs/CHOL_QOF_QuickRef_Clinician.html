Lipid-Lowering Treatment Pathway

1st Line — High-Intensity Statin

Recommended for all patients with established CVD for secondary prevention (NICE NG238).

  • Atorvastatin 40–80 mg daily
  • Rosuvastatin 20–40 mg daily
Atorvastatin 40 mg Atorvastatin 80 mg Rosuvastatin 20 mg
2nd Line — Statin Declined / Not Tolerated

Document reason. Initiate oral non-statin therapy.

  • Ezetimibe 10 mg daily — first choice
  • Add bempedoic acid if ezetimibe insufficient (NICE TA694)
Ezetimibe 10 mg Bempedoic acid / ezetimibe
3rd Line — Injectable Therapies

Specialist initiation only — refer via agreed local pathway for persistently elevated LDL despite oral therapy. Do not initiate in primary care.

  • Inclisiran — if LDL ≥2.6 mmol/L (NICE TA733)
  • Alirocumab / evolocumab — if LDL >3.5–4.0 mmol/L (NICE TA393 / TA394)
Inclisiran Alirocumab Evolocumab
4

Medication Review — Pharmacist

💊  CHOL003 — Statin / Lipid-Lowering Therapy
  • Confirm statin is on the active medication list — not stopped or marked as not taken
  • If no lipid-lowering therapy prescribed and no exception recorded — initiate or escalate to GP
  • Where statin declined or not tolerated — code reason and initiate alternative therapy
  • Review LFTs annually in patients on statins
  • Assess for myalgia and other statin side effects at every review
🧪 Blood tests to arrange — fasting lipid profile (total cholesterol, LDL, HDL, non-HDL, triglycerides) + liver function tests (annual statin monitoring). Renal function if on ACE-I, ARB or diuretic.
🔄 After any therapy change — arrange a repeat fasting lipid profile in 8–12 weeks.
