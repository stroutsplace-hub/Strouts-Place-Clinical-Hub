3

BP Result Pathway

✅  At target
  • Code BP in EMIS same day
  • Proceed to Pharmacist med review as usual
  • QOF indicator met
⚠️  Above target — <180/120
  • Code BP same day
  • Flag to Pharmacist on the day
  • Admin books TC or F2F with Pharmacist within 1 week
  • Pharmacist reviews regimen, up-titrates / adds agent, arranges monitoring (U&E, LFTs, urine ACR)
  • Pharmacist BP clinic follow-up booked after medication change
🚨  BP ≥180/120
  • Do not book routine Pharmacist follow-up
  • Inform GP same day
  • GP assesses for hypertensive urgency / emergency
  • GP determines management and follow-up plan
4

Medication Review — Pharmacist

💊  CHD005 — Antiplatelet / Anticoagulant
  • Confirm medication is actively taken — ask the patient directly
  • If not on therapy and no contraindication — initiate or escalate to GP
  • Stable CHD: aspirin 75 mg first line; clopidogrel 75 mg if aspirin contraindicated/not tolerated
  • Post-ACS/PCI: DAPT may be in place — confirm duration with cardiology letters. Should revert to monotherapy at 6–12 months
  • Review PPI gastroprotection — prescribe if not already in place, especially in older patients
📋  LES — High-intensity Statin
  • All CVD patients (CHD, stroke/TIA, PAD) should be on a high-intensity statin
  • Check current statin and dose at every medication review
  • Up-titrate or switch if not already on a qualifying statin
  • Document reason if high-intensity statin not prescribed (intolerance, decline, contraindication)
  • Consider alternative lipid-lowering agent if statin not tolerated; escalate to GP if uncertain
Atorvastatin 40 mg ✓ Atorvastatin 80 mg ✓ Rosuvastatin 20 mg ✓
🧪 Monitoring investigations — arrange as appropriate following medication initiation or dose change: renal function (U&E), liver function (LFTs), urine ACR. Particularly important after starting/up-titrating ACE inhibitors, ARBs, diuretics, or statins.
