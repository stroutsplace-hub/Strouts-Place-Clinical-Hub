Non-Diabetic Hyperglycaemia QOF 2025/26
For: GP / Pharmacist / HCA / Nurse Version 1.0 · April 2025 Owner: GP Lead
📊 QOF Indicator at a Glance
Code What to achieve Pts Target Key condition
NDH002 HbA1c or fasting blood glucose performed in the preceding 12 months 18 50–90% All patients on NDH register aged ≥18
ℹ️
Register definition: Patients aged 18+ with a coded record of non-diabetic hyperglycaemia or pre-diabetes not superseded by a diabetes diagnosis prior to the start of the financial year. Either an HbA1c or fasting blood glucose satisfies the indicator — both are not required.
🩸 Diagnostic Thresholds — NDH vs Diabetes
Normal
HbA1c <42
mmol/mol
or FPG <5.5 mmol/l — discuss with GP whether to maintain on NDH register
NDH / Prediabetes
HbA1c 42–47
mmol/mol
or FPG 5.5–6.9 mmol/l — code result, reinforce lifestyle advice, offer NHS DPP referral
Diabetes
HbA1c ≥48
mmol/mol
or FPG ≥7.0 mmol/l — flag to GP or pharmacist immediately; remove from NDH register; add to diabetes register
⚠️
Two readings required for formal diabetes diagnosis in asymptomatic patients (NICE PH38 / NG28). One abnormal result should prompt clinical review and repeat testing — do not add to the diabetes register on a single result without GP review unless symptomatic.
🩺 Annual Review — What to Complete
🔄 All NDH Patients — Annual Blood Test
HbA1c or fasting blood glucose (NDH002)
Brief lifestyle advice — diet, activity, weight
Code result with correct SNOMED code same day
Review result and interpret using thresholds above
📋
Code all results on the same day using the EMIS NDH / pre-diabetes template. Log next recall date.
🔗 Also Consider at Each Review
  • Offer or reinforce NHS DPP referral if eligible and not enrolled
  • Review weight and BMI — refer to weight management services if indicated
  • Smoking status — brief advice / cessation referral if current smoker
  • Alcohol status — brief advice if AUDIT-C positive
  • Review medications associated with hyperglycaemia (e.g. steroids, antipsychotics)
⚠️
Progression to diabetes: flag to GP or pharmacist immediately on the same day. Do not leave for the next routine appointment.
🏃 NHS Diabetes Prevention Programme — Referral Eligibility
✅ Eligible for Referral
  • Aged 18 or over
  • NDH blood test result in the preceding 12 months
  • History of Gestational Diabetes Mellitus (GDM) — also eligible for self-referral
🚫 Not Eligible
  • Pregnant
  • Ever diagnosed with type 2 diabetes
  • Moderate or severe frailty
  • Active eating disorder
  • Bariatric surgery within the previous 2 years
💡
Programme completion (>60% of sessions) is associated with a 37% relative risk reduction in developing type 2 diabetes. Digital and face-to-face options are available. Code the referral in EMIS on the day it is made.
🔬 Which Test to Use?
HbA1c — First Choice for Most Patients
  • Non-fasting — convenient for patients
  • Reflects average blood glucose over ~3 months
  • SNOMED: code numeric result in mmol/mol
Fasting Blood Glucose — Use When HbA1c is Unreliable
  • Haemoglobin disorders (e.g. sickle cell, thalassaemia)
  • Haemolytic anaemia
  • Recent blood transfusion (<3 months)
  • Patient must fast for ≥8 hours prior to test
  • SNOMED: code as fasting plasma glucose in mmol/l
⚡ At-a-Glance Decision Aid
Patient has overdue annual test?
  • Request HbA1c (first choice) or fasting blood glucose
  • Book phlebotomy or nurse appointment
  • Code result same day on EMIS NDH template
Result in NDH range (HbA1c 42–47 or FPG 5.5–6.9)?
  • Reinforce lifestyle advice (diet, weight, activity)
  • Offer NHS DPP referral if eligible and not already enrolled
  • Book next annual recall — log date in patient record
Result indicates diabetes (HbA1c ≥48 or FPG ≥7.0)?
  • Flag to GP or pharmacist same day
  • GP to confirm diagnosis (second reading if asymptomatic)
  • Remove from NDH register; add to diabetes register
  • Initiate diabetes management pathway
Result normal (HbA1c <42 or FPG <5.5)?
  • Code result
  • Discuss with GP — should patient remain on NDH register?
  • Document decision in patient notes
  • If removed from register: confirm correct SNOMED coding
