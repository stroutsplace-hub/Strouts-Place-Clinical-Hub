📋 QOF Indicators at a Glance
Code Indicator Points Target Window
AST012 New diagnosis of asthma (on or after 1 April 2025) with a record of an objective test between 3 months before or 3 months after diagnosis 15 pts 45–80% ±3 months of diagnosis
AST007 Annual review including: asthma control (ACT), number of exacerbations, inhaler technique, and written personalised action plan — all coded same day 20 pts 45–70% Preceding 12 months
⚠️
All four AST007 components must be coded on the same day as the review — coding any element on a different date means the indicator is not met.
🔬 New Diagnosis — Objective Testing Pathway (AST012)
Adults & Young People (>16 years)
1
FeNO or blood eosinophil count
Diagnose if FeNO ≥50 ppb or eosinophils above lab reference range
2
BDR spirometry
If step 1 inconclusive. Diagnose if FEV1 increase ≥12% & ≥200 ml or ≥10% predicted normal
3
PEF variability
If spirometry unavailable. Twice daily for 2 weeks. Diagnose if variability ≥20%
4
Bronchial challenge test
Refer if still suspected and all above inconclusive
Children & Young People (5–16 years)
1
FeNO
Diagnose if FeNO ≥35 ppb
2
BDR spirometry
If FeNO not raised or unavailable. FEV1 increase ≥12% from baseline or ≥10% predicted
3
PEF variability
Twice daily for 2 weeks. Diagnose if variability ≥20%
4
Skin prick / IgE + eosinophils
If inconclusive. Diagnose if sensitised or IgE raised and eosinophils >0.5×10⁹/L
5
Bronchial challenge (paediatric specialist)
Refer if still uncertain
💡
Any test in the sequence counts for AST012. If objective testing is not feasible, treat empirically with ICS and retry every 6–12 months. Apply PCA exception code where patient declines or testing is not appropriate.
📝 Annual Review Components (AST007)

Use (RP) LTC Review Template CEG v1.4 in EMIS. All four components must be coded on the same day.

📊 1. Asthma Control — ACT
  • Complete the Asthma Control Test (ACT) with the patient
  • Score out of 25 — code result and date on day of review
  • See ACT scoring guide below
🗓️ 2. Number of Exacerbations
  • Record number of exacerbations in the preceding period
  • Must be coded on the same day as the review
  • Include OCS courses and unscheduled attendances
💨 3. Inhaler Technique
  • Assess technique and document whether satisfactory
  • Record any corrective advice given
  • Code on the day of the review appointment
📄 4. Personalised Action Plan
  • Provide or update written personalised asthma action plan
  • Review and update — do not carry forward an out-of-date plan
  • Code provision on the day of the review appointment
📊 Asthma Control Test (ACT) — Scoring Guide

5 questions, each scored 1–5. Maximum score = 25. Higher score = better control.

25
Fully Controlled

No action required. Maintain current treatment.

20–24
Well Controlled

Continue current treatment. Review at next annual appointment.

16–19
Partially Controlled

Review adherence and inhaler technique before stepping up. Consider triggers.

≤15
Uncontrolled

Flag to GP. Review treatment, adherence, and technique. Consider step-up.

⚠️
Before stepping up treatment, always check adherence using prescription records and reassess inhaler technique — poor adherence and technique are the most common causes of apparent uncontrolled asthma.
✅ Best Practice at Every Review
⚠️ SABA Rule
  • SABAs must never be prescribed without a concomitant ICS (NG245 1.6.3)
  • If identified: flag to Pharmacist for review
📋 Adherence Check
  • Check adherence via prescription records
  • Explore non-adherence before stepping up treatment
🌿 Triggers
  • Smoking status
  • Occupational exposures
  • Seasonal / environmental factors
  • Psychosocial factors
🔁 Referral Thresholds
Aged ≥12 years — Refer to Asthma Specialist
  • Asthma not controlled on high-dose ICS treatment (NG245 1.7.11)
  • Diagnostic uncertainty despite objective testing
  • Suspected occupational asthma — refer to occupational asthma specialist
Aged 5–11 years — Refer to Paediatric Specialist
  • Not controlled on paediatric moderate-dose MART or ICS/LABA ± LTRA (NG245 1.8.7)
  • Diagnostic uncertainty — consider bronchial challenge test via specialist
💡
Pharmacological management algorithms: Algorithm C (≥12 years) and Algorithm D (5–11 years) — available at nice.org.uk/guidance/ng245/resources
