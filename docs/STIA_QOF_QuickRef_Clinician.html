BP Result Pathway

✅  At target
  • Code BP in EMIS same day
  • Proceed to Pharmacist medication review
  • QOF indicator met
⚠️  Above target — <180/120
  • Code BP same day
  • Flag to Pharmacist on the day
  • Admin books TC or F2F with Pharmacist
  • Pharmacist reviews regimen; up-titrates or adds agent
  • Arrange monitoring (U&E, LFTs) as appropriate
  • Pharmacist BP clinic follow-up after medication change
🚨  BP ≥180/120
  • Do not book routine Pharmacist follow-up
  • Inform GP same day
  • GP assesses for hypertensive urgency / emergency
  • GP determines management and follow-up plan

Antiplatelet & Anticoagulant Guidance (STIA007)

🟠  Non-AF — First line

Clopidogrel 75 mg daily is the preferred long-term agent (note: unlicensed for TIA but recommended by NICE NG128 / BNF).

  • Reduces risk of further vascular events by ~25%
  • Use for non-haemorrhagic stroke and TIA
⚠️  Non-AF — If clopidogrel contra-indicated / not tolerated
  • MR dipyridamole + aspirin
  • If both aspirin & clopidogrel contra-indicated: MR dipyridamole alone
  • If MR dipyridamole & clopidogrel both contra-indicated: aspirin alone
💙  Stroke associated with AF

Review for long-term anticoagulation — warfarin or a DOAC (apixaban, rivaroxaban, edoxaban, dabigatran).

  • Confirm anticoagulant is actively being taken
  • Check INR is in range where warfarin is used
  • If anticoagulation not yet initiated — refer to local anticoagulation service for initiation
  • Refer to AF SOP if anticoagulation is suboptimal
⚠️ STIA007 register scope: This indicator applies only to patients coded as non-haemorrhagic (ischaemic) stroke or TIA. Haemorrhagic stroke patients are excluded from the denominator — verify coding rather than exception-reporting.

Medication Review — Pharmacist

💊  STIA007 — Antiplatelet / Anticoagulant
  • Confirm medication is actively taken — ask the patient directly
  • If not on therapy and no contraindication — review reasoning and escalate to GP
  • Confirm the correct agent is prescribed per NICE NG128 / BNF guidance
  • For AF patients: confirm DOAC or warfarin prescribed and being taken; check INR if on warfarin; if not yet initiated refer to local anticoagulation service
  • Code STIA007 once confirmed as taken at this appointment
📋  Statin Therapy (CHOL003 / CHOL004)
  • Stroke/TIA patients are in scope for CHOL003 (statin prescribing) and CHOL004 (LDL ≤2.0 or non-HDL ≤2.6 mmol/L)
  • Confirm a statin is prescribed — high-intensity preferred per LES
  • Up-titrate or switch if not on a qualifying statin
  • Document reason if high-intensity statin not prescribed
  • Consider alternative lipid-lowering agent if statin not tolerated
Atorvastatin 40 mg ✓ Atorvastatin 80 mg ✓ Rosuvastatin 20 mg ✓
🧪 Monitoring investigations — arrange as appropriate following medication initiation or dose change: renal function (U&E), liver function (LFTs). Particularly important after starting or up-titrating antihypertensives, anticoagulants, or statins.
6

Coding Reminders

🩺  BP Readings
  • Code all BP readings in EMIS on the day of the appointment
  • Distinguish clinic readings from HBPM readings using correct SNOMED codes
  • Verify age band at each review — apply STIA014 (≤79) or STIA015 (≥80) correctly
💊  STIA007 — Medication Coding
  • Code medication as being taken — not merely prescribed
  • Confirmation should follow direct discussion with patient at medication review
  • Verify stroke type is coded as non-haemorrhagic — determines STIA007 eligibility
  • Secondary care correspondence: admin to scan and code medication changes on receipt
