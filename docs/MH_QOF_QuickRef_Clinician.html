💡
MH011 high-risk (12-month window): Currently prescribed antipsychotics AND/OR pre-existing CVD (CHD, diabetes, stroke, PAD, CKD) AND/OR current smoker AND/OR BMI ≥23 (≥25 if White ethnicity). All others: 24 months.
🩺 One-Stop Clinic — HCA (30 min)

All checks & bloods in one visit. Code same day using (RP) LTC Review Tower Hamlets CEG v1.4 template.

  • ✓Blood pressure (MH003)
  • ✓Weight & BMI (MH006)
  • ✓Waist circumference if BMI ≥25
  • ✓AUDIT-C alcohol screen (MH007)
  • ✓Lipid profile — confirm 12m or 24m window (MH011)
  • ✓Blood glucose / HbA1c — unless diabetic (MH012)
  • ✓All results coded same day in EMIS
💊 Pharmacist — Follow-up TC Appointment

Conducted once One-Stop Clinic blood results are available. The TC covers three areas:

📄 Care Plan (MH002)

Review, update and agree the care plan with the patient and/or carer. Ensure it is coded in EMIS.

🧪 LES Physical Health Check
HbA1c
HDL cholesterol
Blood pressure
BMI
💊 Medication Review

Antipsychotic efficacy, side effects, adherence and optimisation.

📋 Code all LES components using the (RP) LTC Review Tower Hamlets CEG v1.4 template in EMIS
⚠️
ECG — not routine. Only if cardiac risk changes (new IHD or arrhythmia). ELFT are responsible for the initial ECG prior to antipsychotic initiation.
